OM MIG

GRUNDLÆGGENDE WEB

TEMA 2

I tema 2 startede vi for alvor med kode og design. Vi fik introduktion til Figma, HTML og CSS, og startede med simple opgaver, såsom 'Hello World'

Senere skulle vi kode vores egen hjemmeside ud fra materiale vi fik udleveret. Her skulle vi følge en bestemt wireframe, og derefter style vores site med farver, en menu og oprette grid i CSS.

RESULTAT

Dette er resultatet af vores første site. Når man kigger tilbage på det, var det måske et interessant farvevalg, men det viser blot hvor langt man er kommet, på så kort tid.

Denne opgave havde stort fokus på at følge udleveret wireframe og få det til at passe i CSS grid.

Du kan se sitet ved at trykke på knappen herunder.
