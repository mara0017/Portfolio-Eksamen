OM MIG

GRUNDLÆGGENDE ANIMATION

TEMA 4

I tema 4 var fokus på animation og vi blev for første gang introduceret til Above Illustrator og JavaScript. Udover JavaScript arbejdede vi også videre på HTML og CSS. Her lærte mere advanceret kodning så som CSS animationer og HTML forms.

Her fik vi en opgave hvor vi skulle finde på et "emergency site". I denne opgave blev der udleveret style tile, HTML skabelon, custom CSS som blot skulle tilpasse, da fokus primært var på animationer.

RESULTAT

Desværre havde jeg noget i mit privatliv som gjorde jeg ikke var til meget af tema 4. Dog fik jeg lavet alle CSS og JavaScript øvelsesopgaver for at få styr på animation.

Derfor har jeg også valgt at tilføje en lille CSS animation til min burgermenu på dette site, samt interaktiv JavaScript på forsiden.
